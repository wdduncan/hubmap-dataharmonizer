CanCOGeN Guidelines for Covid19 Biosample metadata collection and submission

Guidelines for submitting metadata regarding Covid19 biosamples and their associated cases to sequence repositories like IRIDA, NCBI Bioproject, and GISAID.

Field Description Guidance Examples Data Status

Identifer
Sample ID A unique value that identifies the sample. None

Experiment Metadata
Project The project that sample was collected for. KPMP, BUKMAP None
Experiment Name The name of the experiment. Depending on the project, expirements are named according to the formats "KPMP_[Date][A-Z]_[assay code]" or "BUKMAP_[Date][A-Z]_[assay code]" KPMP_20210917B_10X-Dual-R, BUKMAP_20210917B_10X-Dual-AC None
Experiment Status The status of the expiremnt. S0 (nuclei), S1 (cDNA), S3 (NGS ongoing), S4 (fastq), S5 (Analysis), S6 (Deposited), GO None
